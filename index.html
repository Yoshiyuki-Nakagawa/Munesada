本同盟の原則

本原則は強制的な義務ではありません。同盟の運営を円滑に行うためのガイドラインです。

総則
  1. 各自の実生活および仕事を最優先して，ホワイトアウト・サバイバルは余裕のある範囲内で遊んでください。

  2. 本同盟と敵対関係にない同盟(不可侵条約を結んだ同盟，友好条約を結んだ同盟，および現時点で交戦状態にない同盟)に属している，プレイヤー領地および施設(同盟旗を含む)への攻撃を禁止します。ただし，サーバーイベントなどで攻撃が許可されるイベントにおいては，本原則よりイベントルールを優先してください。なお，誤って攻撃してしまった場合は，相手の同盟と交渉・謝罪する必要がありますので，盟主または R4 メンバー(幹部)に相談してください。

  3. 世界チャットでは，軽口程度は問題ありませんが，軽口も人によってどのように解釈されるかを考えて，人格攻撃や差別的表現など，一般社会でも明らかに不適切と見なされる言動は謹んでください。世界チャットでは同盟名も併記されますので，たとえ個人の見解であっても同盟の総意であると誤認される危険性があります。

  4. 本同盟のメンバーランクや他同盟の所属者であるかどうかに関係なく，人として対等であることを理解し，互いに尊重しあってください。正義感や美学が対立するときは，まず話し合いで解決を試みてください。仲裁が必要であれば，盟主または幹部に相談してください。

  5. これが最も難しいのですが，怒りの制御(アンガー・マネジメント)をお願いします。多数のプレイヤーで遊ぶゲームなので，納得しがたい言動を発するプレイヤーや，悪意あるように見える行動をするプレイヤーも存在します。短絡的な行動に出ないでください。怒りが収まらなければ，まず盟主または幹部の誰かに相談してください。

  6. 本同盟に不安や不満がございましたら，同盟脱退の自由を認めます。ただ，その前に盟主または幹部の誰かに相談していただければ，可能な範囲で改善を試みます。

  7. 原則はあくまでも原則であり，緊急事態が発生した場合は必要に応じて現場の判断で動いて構いません。ただ，事後で結構ですから，いずれかの幹部，できれば盟主への報告をお願いします。

R2 メンバー
  1. 可能な範囲で，領地の勢力が 1000000 (1M) を目標に成長してください。1M を超えたことに盟主または幹部の誰かが気付き次第，R3 メンバーに昇格設定します。

  2. 同盟科学技術への寄付は，デイリーミッションの報酬が出る 40 回までで十分です。それ以上の任意の寄付には感謝いたします。資源は勢力の成長に使ってください。

  3. 同盟メニューの同盟戦争ボタンから，自動加入を設定して，それを有効化してください。自動加入の設定方法は，豆知識をご覧ください。

R3 メンバー
  1. モンスター討伐の発起など，ささやかで結構ですので R2 メンバーの成長を支援するイベントを開催してあげてください。

  2. 余力の範囲内で結構ですから，同盟科学技術への寄付を多めにお願いします。

  3. 戦争勃発時には，余力の範囲内で R2 メンバーに援軍を出してください。

R4 メンバー：幹部
  1. R3 メンバーへのお願いは引き続きお願いします。

  2. できることが多いため，総則にもありますように，時間的な無理は絶対にしないでください。

  3. 幹部として，余裕があるときで結構ですから，以下の対応をしていただけると助かります。
    1. 加入した新メンバーを R2 にランクアップさせることをお願いします。

    2. 総力が 1M を超えた R2 メンバーを R3 にランクアップさせることをお願いします。新メンバーで既に 1M に達しているメンバーも同様です。

    3. 十分な資源を寄付していただいた同盟科学技術の研究を開始して，同盟の強化をお願いします。

    4. 盟主不在のときに同盟内で揉めごとが発生した場合，中立の仲介者として調停を試みることをお願いします。判断しかねる場合は，当事者には揉めごとの一時停止をお願いして，盟主に相談してください。

    5. どの同盟にも所属していない同盟資源地を獲得するように，同盟旗を建造してください。他の同盟の領地を横切ることは，システム的には容易にできますが，横切られる同盟と交渉しないと，誤解やトラブルが発生する可能性がありますので，予め盟主に相談してください。

    6. 盟主が 3 日以上不在になる場合は，システムから自動で同盟メンバーに対して脱退を促そうと解釈されるパネルが出てしまいます。盟主が 3 日以上不在になることが事前に分かっている場合は，盟主を含む幹部会議で相談の上，いずれかの幹部が臨時盟主になってください。

    7. 盟主が突然の入院などで，不在理由を告げられないまま 3 日以上不在になってしまった場合は，速やかに幹部会議を開いて，幹部を含む同盟メンバー全員の中から次代の盟主を決定し，了承を得てください。冷静な対応ができるメンバーがお勧めです。なお，盟主交代の際には，システムルールとしていずれかの幹部がダイヤを 200 個負担する必要があります。必要なダイヤの負担者については，幹部会議で相談してください。そして，まず負担者の幹部が盟主となり，その後，承諾して下さったプレイヤーに盟主を譲渡してください。

R5 メンバー：盟主
  1. R3 メンバーおよび R4 メンバーへのお願いも引き続きお願いします。

  2. 盟主は他同盟と交渉，必要と判断したら謝罪を心がけてください。対外交渉ではアンガー・マネジメントを強く心がけてください。

  3. 他同盟と交渉するときは，まずはその同盟の盟主に直接個人チャットで交渉してください。相手の盟主に連絡がつかないときは，世界チャットで交渉したい旨を告げて交渉窓口のメンバーを問い，その方と個人チャットで交渉してください。

  4. 盟主が仕事に忙殺されたり，事故や病気などで 3 日以上不在にする場合は，以下の対応をお願いします。
    1. 対外窓口の継続のために不在にせざるを得ない理由を述べ，幹部会議で相談の上，臨時盟主を立ててください。

    2. 相談するときに，強制と受け止められる発言は謹んでください。

罰則
  1. ホワイトアウト・サバイバルはゲームの動きが社会の動きよりも速いために，長期の不在は同盟戦力の低下に繋がり，同盟メンバーへの負担が増えてしまいます。そのため，盟主または幹部への事前連絡もなく，21日間(3週間)以上不在の場合は，盟主および幹部で相談の上，除名を検討することがあります。この罰則は，盟主や幹部にも適用されます。長期不在の事前連絡がある場合は，その限りではありません。

  2. 総則にありますように，敵対していない他の同盟メンバーの領地への攻撃を行った場合，代わって謝罪を行うことになる盟主，および幹部で相談の結果，故意または再三の違反と幹部会議で過半数の判断を得た場合，除名することがあります。

附則
  1. 原則の流布は HTML または PDF で公開し，URL を同盟通知で告知してください。そのとき，半角文字で記述すると，URL と判断されたら部分的に伏せられてしまうために，URL は全角文字で記述してください。長い場合は x.gd などの短縮サービスをご利用ください。

  2. 原則に追加や変更を行う場合は，同盟アンケートを取り，同盟全メンバーの過半数を獲得してから行ってください。
なお，本原則は本同盟の運営指針であり，現実世界における法的義務や責任を生じさせるものではありません。
